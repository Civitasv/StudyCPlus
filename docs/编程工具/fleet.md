基于 JVM 构建，跨平台。

## 多光标

`ctrl+shift+uparrow/downarrow`

## 注释

`ctrl+/`

## Move Line up and Down

`alt+shift+uparrow/downarrow`

## Search and Navigate

1. `Ctrl+k`: goto 弹窗，默认搜索文件和symbol
2. `Ctrl+Shift+p`: 仅搜索symbols
3. `Ctrl+p`: 仅搜索文件
4. `Ctrl+Shift+O`: 仅搜索当前文件中的symbol

`Ctrl+Shift+k`: 搜索Actions
`Ctrl+Shift+f`: 搜索字符串
